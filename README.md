## âš™ï¸ Working Principle
1. The webcam captures live video frames.
2. Frames are processed using OpenCV.
3. The system detects hand/fingertip using color detection.
4. The fingertip coordinates are tracked frame-by-frame.
5. The tracked points are plotted onto a virtual canvas.
6. Continuous tracking creates a drawing effect.
